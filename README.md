# Pixel Playground

Pixel Playground is a simple browser-based drawing app built using HTML5 Canvas, JavaScript, and CSS. Users can draw freely, select different colors, and clear the canvas â€” like a mini MS Paint!


## Features

- Draw using mouse
- Pick different colors using color picker
- Clear canvas with one click
- Responsive and lightweight


## How to Run

1. Clone the repo or download the files
2. Open `index.html` in your browser
3. Start drawing!


##  Built With

- HTML5 Canvas API
- Vanilla JavaScript
- CSS3
